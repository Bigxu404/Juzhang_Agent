## 第十二章：技能系统 ​

### 12.1 SkillTool入口 ​

bash

```
head -40 tools/SkillTool/SkillTool.tsx
```

技能系统允许用户通过`/skillname`调用预定义的多步骤工作流。

### 12.2 内置技能 ​

bash

```
ls skills/bundled/
```

包括：commit（提交代码）、loop（循环执行）、schedule（定时任务）、simplify（简化代码）、updateConfig（更新配置）等。

### 12.3 技能预算 ​

技能加载有Token预算限制——约占上下文窗口的1%（约8000字符），防止技能描述占用过多上下文空间。
