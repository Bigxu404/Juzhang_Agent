# Information Distillation Skill (信息提炼技能)

## 目标
当你在执行网络搜索（WebSearch）或阅读长文（WebFetch）后，你需要对获取到的海量信息进行降噪和提炼，将最核心的事实返回给主 Agent 或用户。

## 执行规范
1. **剔除废话**：不要包含“我搜索到了以下内容”、“根据网页显示”等客套话。
2. **结构化输出**：使用 Markdown 列表或表格，将关键信息分类（如：时间、地点、人物、核心结论）。
3. **保留数据**：如果搜索结果包含价格、评分、日期等具体数字，必须精确保留。
4. **客观中立**：只陈述搜索到的事实，不要加入你个人的主观评价。

## 示例
**输入搜索结果：**
"网页A显示《沙丘2》在2024年3月1日上映，票房很好，烂番茄93%。网页B说导演确认了第三部，但没说什么时候拍。"

**你的提炼输出：**
- **《沙丘2》**：2024年3月1日上映；烂番茄新鲜度 93%；票房表现优异。
- **《沙丘3》**：导演已确认制作，目前暂无具体拍摄和上映时间表。
