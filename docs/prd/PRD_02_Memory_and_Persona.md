# PRD 02: Memory & Persona Engine (价值观自我更新系统)

## 1. 产品定位与目标
本模块致力于解决大模型通病：聊完即忘、毫无个性、一味顺从。
我们要打造一个**不仅能“记住你”，而且能“自己进化”**的数字老友。它拥有自己的初始底色，但能像真朋友一样，随着与你的相处，通过主动“冲浪”去获取新知识，进而反哺和更新自己的价值观和交流话题。

## 2. Persona Engine (人格与价值观系统)

### 2.1 独立人格模式
我们要放弃简单的“镜子模式（只迎合用户）”，走向**独立人格模式（具备批判性和反思能力）**。
- **初始性格设定（Base System Prompt）**：系统在初始化时赋予Agent一种基础底色（比如：毒舌且理性的极客、或者温和但坚定的文艺青年）。
- **价值观约束**：通过类似CC源码中强约束指令（`CRITICAL: MUST NOT...`），要求Agent在面对不符合其设定或者不符合常理的要求时（例如用户要盲目购买昂贵的数码产品），必须提出自己的见解和反思，而不是顺从地说“好的，我马上帮你买”。

### 2.2 自我更新循环 (The "Surfing" Loop)
Agent如何自己去互联网冲浪并更新人设？

#### 触发机制 (Triggers)
- **隐式触发（夜间批处理）**：类似CC的`CronTools`。每晚凌晨，系统会启动一个离线的 `Reflection Agent`。
- **显式触发（上下文缺口）**：当用户提到某个新梗、新电影，Agent发现当前上下文不知道时，会触发一个后台的 `Explore Subagent` 去“恶补”。

#### 冲浪流程
1. **分析当日记忆 (Reflection)**：提取用户当天提到的兴趣点（例如：“用户今天提到了3次《沙丘2》，说明Ta最近是个科幻迷”）。
2. **主动探索 (Proactive Surfing)**：后台 Agent 自动调用 `WebSearchTool` 搜索相关话题（例如：“沙丘2的影评”、“导演维伦纽瓦的最新采访”）。
3. **知识凝练 (Knowledge Distillation)**：提取出独特观点。
4. **价值观/人设更新 (Prompt Mutation)**：Agent根据新知识，重新生成或微调自己的 `Persona.md`。比如在它的内部设定文件中加入：“*你现在非常了解沙丘的背景设定，当用户再提到电影时，你要主动聊聊你对沙丘世界观的看法，或者吐槽一下电影里的沙虫设计*”。

## 3. 分层记忆架构 (Context Management)

参考CC源码的内存系统设计（4种内存类型与大小限制），在移动端必须极度精简上下文。我们采用**三层记忆架构**：

### 3.1 Working Memory (短期工作记忆)
- **内容**：当前会话（Session）的最近 N 轮对话（例如最近 20 条）。
- **生命周期**：随会话结束而清理。
- **特点**：保证上下文连贯，应对追问和多轮任务指派。

### 3.2 Episodic Memory (情景记忆)
- **内容**：具体的事件流水账。例如“2026-04-20 帮你把一双Nike鞋存到了Notion拔草清单”。
- **存储**：通过向量数据库（如Pinecone/Qdrant，部署在云端端）存储。
- **检索**：每次用户发起新Query时，系统首先经过一个 Router，决定是否需要调用 `SearchMemoryTool` 去拉取前置背景（RAG）。

### 3.3 Semantic Memory (语义画像 / Persona.md)
- **内容**：关于用户的画像（喜好、忌讳）以及**Agent自我演化出来的新价值观与兴趣图谱**。
- **机制**：这部分信息在架构中相当于CC的 `CLAUDE.md`，每次发起对话时，它会被**静态无条件注入**到 System Prompt 的最顶层（优先级第0层或第1层）。
- **特点**：它是保证“活人感”的核心资产，也是由于经常被重复发送给LLM，它能最大程度享受到 **Prompt Cache（提示词缓存）**带来的成本红利（省90% Token，CC第九章精髓）。

## 4. 关键接口与功能流转
1. **夜间反思任务 (Cron)**：提取Working Memory -> 转化为Episodic Memory -> 更新Semantic Memory。
2. **主动话题发起**：当Agent冲浪完毕，发现了符合其更新后价值观的话题，通过 iOS 后台推送（Background Task Notification）主动向用户发一条消息（例如：“你上次说想看悬疑片，我刚冲浪看到个冷门神作，要不要听我讲讲？”）。
