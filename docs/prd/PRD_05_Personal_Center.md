# PRD 05: 个人中心与全局设置 (Personal Center & Settings)

## 1. 背景与目标
随着移动端 Agent 架构的完善，系统不再提供默认的 API Key（避免公用 Key 导致的 Token 耗尽和隐私问题），Agent 的能力将完全由用户自带的“大脑（API Key）”驱动。
因此，原有的简单“设置页”需要升级为一个完整的**「个人中心」**。它不仅是配置模型参数的地方，更是用户管理 Agent “记忆”、“人设”、“技能”和“外部授权”的核心枢纽。

**核心目标：**
1. **大脑配置**：强制用户配置自有 API Key，支持多模型切换。
2. **记忆透明化**：让 Agent 的“潜意识”和“长期记忆”对用户可见、可控。
3. **能力扩展**：提供技能（Skills）和工具（Tools/MCP）的管理入口。
4. **隐私与授权**：集中管理外部应用（如 Notion）的授权状态。

---

## 2. 核心功能模块拆解

### 2.1 账号与基础档案 (Account & Profile)
*   **用户档案**：头像、昵称、唯一 UUID（支持后续社交图谱扩展）。
*   **账号绑定**：Apple ID 绑定、手机号绑定（便于跨设备同步）。
*   **数据导出与注销**：支持一键导出所有聊天记录和记忆库，支持彻底注销账号清空云端数据。

### 2.2 大脑与模型配置 (AI Core Configuration)
*   **API Key 管理**：
    *   支持配置多个厂商的 Key（如 MiniMax, OpenAI, Anthropic, DeepSeek）。
    *   前端掩码显示（`sk-****`），支持验证 Key 的连通性。
*   **主模型选择 (Main Agent Model)**：
    *   用于日常对话和高情商回复，用户可从下拉列表中选择（如 `abab6.5s-chat`, `gpt-4o`）。
*   **探索模型选择 (Explore Subagent Model)**：
    *   用于后台静默冲浪和长文总结，推荐用户选择速度快、成本低的小模型（如 `gpt-4o-mini`, `claude-3-haiku`）。

### 2.3 记忆与人设管理 (Memory & Persona) - *核心差异化功能*
*   **当前人设看板 (Current Persona)**：
    *   展示 Agent 目前对用户的认知（例如：“你认为我是一个喜欢科幻电影、说话直接的程序员”）。
    *   支持用户手动编辑或纠正人设。
*   **记忆保险库 (Memory Vault)**：
    *   列表展示向量数据库（Episodic Memory）中存储的记忆碎片。
    *   支持按时间、关键词搜索记忆。
    *   **遗忘机制**：支持用户左滑删除某条特定记忆（“忘掉这件事”）。
*   **记忆重置**：一键清空所有长期记忆，让 Agent 恢复“出厂设置”。

### 2.4 技能与工具库 (Skills & Tools)
*   **我的技能 (My Skills)**：
    *   展示当前已激活的 Prompt SOP（如“信息提炼”、“小红书文案生成”）。
    *   支持用户自己编写并上传自定义 Skill（定义触发词和 SOP 流程）。
*   **外部授权管理 (MCP & Integrations)**：
    *   集中展示所有已授权的外部平台（如 Notion, Google Calendar）。
    *   支持一键撤销授权（Revoke Access）。

### 2.5 交互与偏好设置 (Preferences)
*   **微表情与震动 (Haptics & UI)**：
    *   开关：打字机效果延迟、状态栏呼吸灯、关键操作的触觉震动反馈。
*   **语音与播报 (Voice - 预留)**：
    *   TTS 声音选择、语音输入自动发送开关。
*   **系统级捕获 (Share Extension)**：
    *   配置离线总结的默认行为（如：分享链接后是“仅总结”还是“总结并存入 Notion”）。

## 4. 异常流设计 (Exception Handling)
1. **未配置 API Key 拦截**：
   - 当用户注册后首次进入聊天界面，若未配置 Key，在发送消息时，系统直接在聊天流中返回系统提示：“请先前往个人中心配置您的 API Key，唤醒我的大脑。”，并阻断后续的 Agent Loop。
2. **API Key 欠费/失效**：
   - 捕获大模型接口返回的 401/402/500 错误，解析后返回给用户：“大脑连接失败，请检查个人中心的 API Key 额度或模型权限。”
