# 附录：概念速查与决策清单

---

## A. 关键概念速查

### Agent 循环
- **AgentStreamEvent**: 向外发射的事件类型（文字/工具/完成/错误）
- **AbortSignal**: 取消执行的信号

### 上下文管理
- **Snip**: 删除早期工具调用对
- **MicroCompact**: 规则去重替换
- **AutoCompact**: 全量 LLM 压缩

### 记忆系统
- **M1 在线上下文**: 内存，当前会话
- **M2 项目记忆**: ALICE.md，项目约定
- **M3 向量记忆**: 向量库，语义历史
- **M5 用户画像**: identity/workflow/voice.md

---

## B. 架构决策清单

在开始设计前确认：
1. **状态管理**：状态住在哪里？生命周期多久？谁读谁写？
2. **工具 vs 服务**：AI 主动触发是用户期望的吗？是则工具，否则服务。
3. **并发安全**：读写是否分离？是否有防递归守卫？
4. **降级路径**：组件挂掉后，系统如何降质运行而不崩溃？
5. **可观测性**：决策是否携带 `reason`？是否有结构化日志？
