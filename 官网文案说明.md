# 官网文案 + 碎片卡库（B1 克制成年人版）

版本：v0.1

> 定位提醒：**朋友式 Agent**（关系先于功能），但不拟人表演；氛围是**黄昏暖灯 + 深胡桃木桌面**；叙事重点是**积累与变化**（不使用时间轴）。

---

## 0. 使用方法（给运营 / 前端 / 设计）

- **官网全页面文案**：直接复制粘贴即可（每段都已控制长度与语气）。
- **碎片卡库**：用于 World 页（世界碎片）、App 内碎片流、空状态提示、轻推送（如有）。
- **一致性规则**：任何新增文案都必须遵守第 1 节的 Copy System。

## 1) Copy System（B1 + 家庭意象）

### 1.1 语气定义

- **克制、可信、长期**：少形容词，多边界与事实。
- **保留家庭意象**：作为“背景光”，不承担情绪结论。

### 1.2 硬规则（必须）

1. **不许诺永恒**：禁用「永远 / 一定 / 绝不」。
2. **不情绪绑架**：禁用「最懂你 / 只有我懂你 / 治愈 / 灵魂伴侣」。
3. **不技术炫耀**：主页面不出现「大模型 / RAG / 多模态 / 训练」。
4. **每屏意象词上限**：每屏最多 1 个意象词（Hero 可 2 个）。

### 1.3 白名单意象词（可用）

- 光与空间：黄昏、灯光、暖光、屋里、桌面、窗边、夜里
- 物件与材质：胡桃木、纸张、便签、杯垫、余温
- 状态与动作：坐下、停一停、留一盏灯、把话放下

### 1.4 句式与长度

- 标题：8–14 字（中文）
- 副标题：20–48 字（两句内）
- 正文：每段最多 2 句
- 列表项：每项 6–14 字

# 9) 碎片卡库（Fragments Library）v1（100 张）

> 用途：World 页卡片、App 内碎片流、空状态、轻提示。

**字段规范**

- 标题：4–10 字
- 内容：1–2 句，总字数 ≤ 28
- 风格：B1（克制成年人）+ 低频意象

## 9.1 桌面与片段（20）

 1. 《桌面便签》：片段可回看，也可删除。
 2. 《杯垫圆环》：边界是默认项，不是附加项。
 3. 《纸张余温》：重要内容会被优先保留。
 4. 《一张折角》：你标记过的，会更容易被找回。
 5. 《不必从头》：重复信息会被压缩成要点。
 6. 《留一行字》：你只写关键句也可以。
 7. 《放回原处》：整理后的内容可被还原与调整。
 8. 《可被取用》：碎片不是收藏，是可用结构。
 9. 《不求完整》：不追求记住一切。
10. 《只留重要》：记忆默认有取舍。
11. 《随手一条》：短记录也能变成结构。
12. 《低噪音》：信息呈现优先清晰。
13. 《可折叠》：不需要的部分可以收起。
14. 《可合并》：重复内容会被合并为更短的版本。
15. 《可回滚》：你可以撤销一次整理。
16. 《轻标记》：你可以用标签管理碎片。
17. 《不抢视线》：碎片只在需要时出现。
18. 《可迁移》：碎片可以被导出带走。
19. 《可清空》：你可以一键清理某类记忆。
20. 《桌面很安静》：系统不默认推送。

## 9.2 日常结构（20）
